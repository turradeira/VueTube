## Features

<img src="./resources/Features.svg" alt="VueTube icon" height="100"/>

- 🎨 Temas: Claro, Escuro, OLED, Todas as cores do arco-íris
- 🖌️ Interface Customizável: Pode customizar por completo a accent color, e  outras partes da interface para remover funcionalidades que não usa!
- ⬆️ Atualizações Automáticas: Seja notificado quando uma atualização está disponivel e desatualize se não gostar!
- 👁️ Proteção de Rastreio: Nenhuma telemetria é envida do seu dispositivo
- 📺 Custom video player
- 👎 Return YouTube Dislike
